# Cryptocurrency Statistics CLI for Node.js

<img src="./cli-image.png" alt="term" width="800">

https://www.npmjs.com/package/cryptostats-cli

## Features
- Table pagination
- Change currency
- Change period
- Real-time data (1 min refresh)

## Requirements
- Node.js. & NPM

## Instructions
```sh
$ npm install -g cryptostats-cli
$ cryptostats
```
